Why Create This?

     A hands on experience of chemical reactions can give an intuitive understanding that just reading about it can't. It can be made as fun as any video game with competitions, puzzels (like in portal), fps elements, and skill progress. Making chemistry enjoyable can change it from a chore to an enjoyable pass time. More people understanding chemistry means more peopls creating things that everyone can benifit from.

The Problem:

     The current version uses the gravity equation to apply a force based on the radius between them. This is obviously not accurate so the first step is to fix this. Molecular Mechanics set up a simple algebraic expression for the total energy of a compound and a prediction of how it would act.

Solving The Problem
But How?

     We can hold the number of protons, neutrons, and electrons in variables, as well as any other number we might need to hold. We can get the distance between each atom and we can apply forces on each atom. What we need is a formula that can take these inputs and output the magnatude and direction of the force.
